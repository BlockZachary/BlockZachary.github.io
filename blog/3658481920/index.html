拉一个 docker 的 mysql

先创建一个文件夹

如：

mkdir docker_env

然后进入这个目录

cd docker_env

在这个目录下，创建一个文件

touch docker-compose.yml

然后在这个文件里面编辑

接着在这个目录下运行

docker-compose up -d

docker 如果很慢的话 配置一下 docker 应用程序的源
