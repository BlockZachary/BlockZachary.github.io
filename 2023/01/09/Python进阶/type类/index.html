Python进阶-type类

type 类

  • 任何 class 在内存中 就是一个 type()类的对象

这种一般是在写源码时还不知道需求，不知道需要具体创建什么样的类，从而在运行时动态创建类出来

  • 可以使用 type()来创建其他 class -> type(class_name, parents, class_dict)
  • 可以使用 type()来动态创建 class
